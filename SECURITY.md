# Security Policy

Please report all security-related issues through
[here](https://github.com/CosmoCreeper/Sine/security/advisories/new).

## Supported Versions

The list below shows which versions of Sine are still supported.
Please note that we plan to have a slightly longer support policy in the future,
supporting versions for at least one month after the next one releases, and we will likely expand from there.

| Version | Supported          |
| ------- | ------------------ |
| 2.3.0   | :white_check_mark: |
| < 2.3   | :x:                |
